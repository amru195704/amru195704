# P3-2 バイリニア補間 — 手計算の期待値

対象: `pysrc/ch03/ch03_bilinear.py`（架空・誇張テストデータ）
検証方式: 手計算（グリッド系のため。読者が自分で期待値を作れることが教材の主眼）

## 入力と格子

- 入力点: 緯度 35.6810、経度 139.7670（東京駅付近）
- 含むメッシュ: 53394611（南西隅ノード = 35.675, 139.7625）
- 格子間隔: 緯度 1/120 度、経度 1/80 度
- 4隅の補正量 [秒]（架空値）: SW(12.0, −8.0) NW(12.4, −8.6) SE(12.2, −8.2) NE(12.8, −9.0)

## 内分比

- t（南北）= (35.6810 − 35.675) / (1/120) = 0.0060 × 120 = **0.72**
- s（東西）= (139.7670 − 139.7625) / (1/80) = 0.0045 × 80 = **0.36**

## 重み（合計1）

- w_SW = (1−t)(1−s) = 0.28 × 0.64 = 0.1792
- w_NW = t(1−s)     = 0.72 × 0.64 = 0.4608
- w_SE = (1−t)s     = 0.28 × 0.36 = 0.1008
- w_NE = t·s        = 0.72 × 0.36 = 0.2592
- 合計 = 1.0000 ✓

## 補間結果 [秒]

- Δ緯度 = 12.0×0.1792 + 12.4×0.4608 + 12.2×0.1008 + 12.8×0.2592
      = 2.1504 + 5.71392 + 1.22976 + 3.31776 = **12.41184 秒**
- Δ経度 = −8.0×0.1792 − 8.6×0.4608 − 8.2×0.1008 − 9.0×0.2592
      = −1.4336 − 3.96288 − 0.82656 − 2.3328 = **−8.55584 秒**

## 距離換算（1秒 ≈ 30.8875 m、経度は ×cos35.681°=0.81206）

- 南北 = 12.41184 × 30.8875 = **383.37 m**
- 東西 = −8.55584 × 30.8875 × 0.81206 = **−214.66 m**
- 水平 = √(383.37² + 214.66²) = **439.38 m**

## 照合

|---|---|---|---|
| Δ緯度 [秒] | 12.41184 | 12.41184 | 0 |
| Δ経度 [秒] | −8.55584 | −8.55584 | 0 |
| 水平距離 [m] | 439.38 | 439.38 | 0 |
